AGDeblend
=========

AGDeblend is an open source library containing a function for blending seismic data (or adjusting already-blended data) and a function for deblending seismic data. In this documentation you will find a description of how AGDeblend works, how to install it, how to use it, a discussion of the provided examples, and an overview of the code structure.
